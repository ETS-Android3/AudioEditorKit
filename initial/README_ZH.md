## 工程目录结构

    |-- com.huawei.audioeditor.demo
        |-- Activity
            |-- MainActivity // Audio Editor UI SDK 入口,格式转换入口,音频提取入口
	        |-- AudioFormatActivity  // Demo格式转换
			|-- AudioFilePickerActivity  // Demo选择音频文件、列表
